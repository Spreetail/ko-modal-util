Actions Example

Demonstration of adding action buttons with callbacks to modal show. 

                ko.Modal.show({
                    title: 'Example 01',
                    template: 'example-modal',
                    actions: [{
                        label: 'Cancel',
                        css: 'btn-danger',
                        onClick: ko.Modal.hide
                    }, {
                        label: 'Save',
                        css: 'btn-success',
                        onClick: function() {
                            alert('User has selected Save from the modal.');
                            ko.Modal.hide();
                        }
                    }]
                });
